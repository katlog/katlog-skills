# katlog-skills

个人收集和创建的 Claude Code Skills。

## 本仓库 Skills

| Skill | 描述 |
|-------|------|
| [tech-diagram](./tech-diagram/) | 技术图表绘制工具，支持架构图、流程图、ER 图、时序图、动画演示等，基于 Mermaid 和 SVG |
| [geektime-downloader](./geektime-downloader/) | 极客时间课程下载与整理，自动识别课程类型（图文/视频），支持视频转录和内容摘要 |
